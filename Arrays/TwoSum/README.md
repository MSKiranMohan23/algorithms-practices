TWO SUM: Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

MAP Example:
let newMap = new Map()

newMap.set('kk',1)
Map(1) {'kk' => 1}
newMap.set('yy',1)
Map(2) {'kk' => 1, 'yy' => 1}
newMap.set('yy',2)
Map(2) {'kk' => 1, 'yy' => 2}
newMap.set('xx',3)
Map(3) {'kk' => 1, 'yy' => 2, 'xx' => 3}
newMap.has('kk')
true
newMap.get('kk')
1
